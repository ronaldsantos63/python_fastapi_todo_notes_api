# Todo notes API

To-do list API written in python and fastAPI

# Tools

- Python 3.9
- Pipenv

# How to run?

- Clone Repo<br>
``` git clone https://github.com/ronaldsantos63/python_fastapi_todo_notes_api.git```
- Run pipenv<br>
``` pipenv install```
- Run API<br>
``` uvicorn main:app --reload```
